# refuse-classification-applet
毕业设计选题，基于springBoot的垃圾分类识别的小程序。

本项目是我的毕业设计，项目分成前台和后台，前台就是小程序，后台是管理系统。

小程序端采用uni-app开发；后端自然是spring全家桶以及各种中间件。
